Fizzbuzz in Coffeescript
=====================

This project consists of writing Fizzbuzz in coffee script.

Technologies used:
------------------

* Coffee Script
* Mocha
* Chai

How to use
-----------

Clone the repository:

```shell
$ git clone git@github.com:leopoldkwok/fizzbuzz_coffeescript.git
```

Change directory:

```shell
$ cd fizzbuzz_coffeescript
```

Run the tests:

```shell
$ mocha
```
